# **Log Analysis of Windows Virtual Machine using Azure and Defender for Endpoint**

In this project, I will create a Windows Virtual Machine and connect to it using RDP. I will then onboard the VM to MDE and create a simple txt file. Finally, will run a basic KQL query in order to observe the log data that the activity created. 

## **Technology Used:**

- Microsoft Azure Virtual Machine
- Microsoft Defender for Endpoint (MDE)
- Remote Desktop Protocol (RDP)
- Powershell
- Kusto Query Language (KQL)
